## 🧭 Overview

A **data-driven cryptocurrency analytics dashboard** built to decode market behavior through structured visualization and statistical signals.

It enables users to **track trends, evaluate volatility, and assess risk-reward dynamics** across multiple cryptocurrencies.

## ✨ Core Features

| Module | Capability |
|-------|------------|
| 🪙 Asset Selection | Dynamic crypto switching |
| 📊 KPI Engine | Volume, Price Extremes, YTD Gain |
| 📈 Trend Analysis | Time-series price tracking |
| 🕯️ Candlestick Engine | Price action + volatility |
| ⚖️ Risk Lens | Risk vs reward evaluation |
| 📉 Volatility Mapping | Scatter-based fluctuation insight |

---

## 🧠 Analytical Stack

- **D3.js** → Interactive visualizations  
- **JavaScript** → Data handling logic  
- **CSV Pipelines** → Lightweight ingestion  
- **Statistical transforms** → Feature engineering
